# Настройка на Android

<Callout type="info" emoji="ℹ️">
  Для работы Outlivion VPN мы рекомендуем приложение **v2rayNG**.
</Callout>

## 1. Скачайте приложение

[Скачать v2rayNG из Google Play](https://play.google.com/store/apps/details?id=com.v2ray.ang)

## 2. Скопируйте ключ

Получите ваш ключ доступа в [Telegram-боте](https://t.me/outlivion_bot) и скопируйте его.

## 3. Импорт ключа

1. Откройте v2rayNG.
2. Нажмите на иконку "плюс" (+) в верхнем углу.
3. Выберите "Импорт профиля из буфера обмена".
4. Нажмите на кнопку V (внизу справа) для подключения.

<Callout type="success">
  Если подключение успешно, вы увидите надпись "Connected" или значок ключа в строке уведомлений.
</Callout>
